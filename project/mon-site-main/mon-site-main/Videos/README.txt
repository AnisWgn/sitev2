Pour faire fonctionner le fond vidéo avec le Konami Code :

1. Téléchargez une vidéo (format MP4)
2. Renommez-la en "initial-d.mp4"
3. Placez-la dans ce dossier Videos

La vidéo sera automatiquement :
- En arrière-plan
- En lecture en boucle
- Muette (pour ne pas interférer avec la musique)
- Redimensionnée pour couvrir tout l'écran

Assurez-vous d'avoir les droits d'utilisation de la vidéo que vous choisissez !
